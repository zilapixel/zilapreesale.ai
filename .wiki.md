# Project Summary
The ZILA project is a modern, AI-powered Web3 meme ecosystem designed to blend artificial intelligence, decentralized infrastructure, and viral culture into a cohesive token economy. It aims to create a scalable digital economy that leverages community-driven governance and innovative distribution models, making it a unique player in the crypto space.

# Project Module Description
The ZILA project consists of several functional modules, including:
- **Navbar**: Provides navigation links to different sections of the website.
- **Hero Section**: Displays the project’s branding and call-to-action buttons.
- **About/Vision**: Details the purpose and goals of the ZILA project.
- **Stats Section**: Shows key metrics regarding token supply and community engagement.
- **Ecosystem Section**: Outlines the components of the ZILA ecosystem.
- **Live Price Chart**: Displays real-time price data for the ZILA token.
- **Airdrop Section**: Information on the token airdrop event.
- **Tokenomics**: Breakdown of the token distribution strategy.
- **Roadmap**: Timeline of project milestones.
- **Partnerships**: Lists key collaborations.
- **Whitepaper**: Provides access to the project’s detailed documentation.
- **Footer**: Contains additional links and project information.

# Directory Tree

# File Description Inventory
- **index.html**: The main HTML document that structures the website.
- **package.json**: Contains project metadata and dependencies for building the project.
- **script.js**: Implements interactive features such as particle animations and modals.
- **style.css**: Contains styles for layout, colors, and responsive design.
- **template_config.json**: Configuration settings for the project template.
- **todo.md**: Development plan outlining design guidelines and file creation tasks.

# Technology Stack
- **HTML**: For structuring the content.
- **CSS**: For styling and layout (with glassmorphism and neon effects).
- **JavaScript**: For dynamic interactivity, using ES6 modules.
- **Vite**: Build tool for modern web projects.
- **Intersection Observer API**: For scroll-triggered animations.

# Usage
1. **Install Dependencies**: Run the command to install dependencies.
