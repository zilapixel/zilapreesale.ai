# ZILA Web3 Website - Development Plan

## Design Guidelines

### Design References
- **Cyberpunk + AI + Web3 aesthetic**
- **Premium crypto project sites** (Solana, Polygon style)

### Color Palette
- Primary Background: #0a0a0f (Deep Dark)
- Secondary Background: #0d1117 (Dark Blue-Black)
- Neon Green: #00ff88
- Neon Cyan: #00e5ff
- White Glow: #ffffff
- Glass: rgba(255,255,255,0.05)
- Card BG: rgba(0,255,136,0.05)

### Typography
- Headings: Orbitron (futuristic)
- Body: Inter (clean, modern)

### Key Styles
- Glassmorphism cards with backdrop-blur
- Neon glow borders and text shadows
- Smooth scroll transitions
- Floating 3D elements with CSS transforms
- Particle animation background (canvas)
- Mouse hover glow effects
